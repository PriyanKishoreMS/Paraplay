# About the Project

A React app that lets you watch videos together from different Streaming platforms including

- File paths
- Youtube
- Facebook
- Twitch
- SoundCloud
- Sreamable
- Vimeo
- Wista
- Mixcloud
- Dailymotion
- Kaltura
